# 餐饮管理系统（Flask学习项目）
目的：通过实现多个API 以此实现逆向学习Flask
## 1.项目目标
实现一个简单的餐饮管理系统，包含用户认证、菜品管理、订单管理等功能

## 2.技术栈
python3.X
Flask
Flask-sqlalchemy（ORM映射）
pyJWT(用户认证)
SQLite（数据库）
